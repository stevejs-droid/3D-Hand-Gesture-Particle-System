# 3D Hand Gesture Particle System

An interactive **3D particle system** controlled using **hand gestures through the camera**.  
This project creates a particle-based visual experience where users can move, rotate, zoom, and scale the scene using simple hand movements.

## 🚀 Features

- 🎥 Camera-based hand gesture control
- ✋ Open hand to zoom in
- ☝ Point / ✊ Fist to zoom out
- ✌ Two fingers to rotate
- 🤏 Pinch gesture to scale up
- 🖐 Move hand to pan the view
- ⬜ White square particle design
- 🌐 Runs best on **HTTPS**
- 👌 Works best with **one clear hand**

## 🛠️ Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **3D particle rendering**
- **Hand gesture detection using camera**

This project was created to explore:

## ▶️ How to Use

1. Open the project in a browser.
2. Allow camera permission.
3. Show your hand clearly in front of the camera.
4. Use gestures to control the particle system:
   - **Open Hand** → Zoom In
   - **Point / Fist** → Zoom Out
   - **Two Fingers** → Rotate
   - **Pinch** → Scale Up
   - **Move Hand** → Pan View
